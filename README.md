# 📊 TechMarket Analytics Portfolio

A portfolio project that demonstrates the complete workflow of a Data Analyst using a real-world e-commerce dataset.

The goal of this repository is not only to analyze data but also to simulate real business tasks faced by analysts in modern e-commerce companies.

---

# 🎯 Project Objectives

- Perform exploratory data analysis (EDA)
- Solve business problems using SQL
- Conduct A/B testing
- Build interactive Power BI dashboards
- Forecast sales
- Deliver business recommendations based on data
