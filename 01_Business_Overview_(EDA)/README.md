# Business Health Check

## Project Overview

The first stage of the TechMarket Analytics Platform focuses on understanding 
and validating the source data before any business analysis begins.

## Objectives

- Load all datasets
- Validate data quality
- Check data consistency
- Identify missing values
- Detect duplicates
- Review data types
- Prepare data for further analysis

## Dataset

Brazilian E-Commerce Public Dataset by Olist.

## Deliverables

- Data Loading Notebook
- Data Quality Report
- Clean dataset for further analysis
